# Note Spec

## Required Sections

- Hook
- Body
- CTA

## Preferred Structure

- Hook: one strong stop-scroll opening
- Body block 1: context or problem
- Body block 2: insight, method, or opinion
- Body block 3: practical takeaway
- CTA: one low-friction next action

## Rules

- Prefer concrete scenarios over generic advice.
- Keep structure scannable.
- Keep CTA aligned with brand goal.
- Keep one main argument only.
- Avoid over-explaining basics the audience already knows.
